# smart-trashcan-engg1101
## Core Idea: 
We create a device that scans trash with AI vision and opens the corresponding bin to make waste sorting easier.

## Unique features that are easy to implement:
- If the lid cannot close, a signal is given (a light turns on, a sound plays, etc.) indicating that the trash bin needs to be emptied/cleared.
- A small announcement before throwing away water bottles/cans, indicating that they should first be emptied of liquids/food before being discarded.

## Unique features that are harder to implement:
- Collect statistics on the weight of trash thrown away over a specific period, what items are most frequently discarded, peak usage times, etc. (any useful statistics).
- If the AI vision is not confident above a certain threshold (e.g., the AI shows 20% paper, 40% metal, 40% plastic), the device either notifies the person to sort the trash into the correct bin themselves or opens a 4th bin for general/unsorted waste.
